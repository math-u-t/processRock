# アイコン画像について

このフォルダには、Chrome拡張機能のアイコン画像を配置します。

## 必要なアイコンサイズ

以下のサイズのPNG画像を作成してください：

1. **icon16.png** - 16x16ピクセル（ツールバー用）
2. **icon48.png** - 48x48ピクセル（拡張機能管理ページ用）
3. **icon128.png** - 128x128ピクセル（Chromeウェブストア用）

## デザインの推奨

- **テーマ**: 盾（🛡️）やロック（🔒）をモチーフにしたシンプルなデザイン
- **カラー**: 紫系のグラデーション（#667eea ～ #764ba2）
- **背景**: 透明またはグラデーション
- **スタイル**: フラットデザイン、モダンな印象

## 簡単な作成方法

### オプション1: オンラインツールを使用

1. **Canva** (https://www.canva.com/)
   - 無料で使えるグラフィックデザインツール
   - テンプレートから簡単に作成可能

2. **Figma** (https://www.figma.com/)
   - プロフェッショナルなデザインツール
   - 無料プランで十分

### オプション2: 絵文字を使用（最も簡単）

1. 絵文字アイコンジェネレーターを使用（例: https://favicon.io/emoji-favicons/）
2. 🛡️（盾）の絵文字を選択
3. 各サイズでダウンロード

### オプション3: Photoshop/GIMPで作成

1. 各サイズのキャンバスを作成
2. 盾や鍵のアイコンを描画
3. 紫系のグラデーションを適用
4. PNGで保存

## アイコンがない場合

アイコン画像がなくても拡張機能は動作しますが、Chrome拡張機能の管理画面で警告が表示されることがあります。

開発中は、デフォルトのアイコン（Chromeのパズルピースアイコン）が表示されます。

---

**ヒント**: とりあえず動作確認したい場合は、アイコンなしでも問題ありません。後から追加することも可能です。
